Website: OnBeginninigs

I chose the article OnBeginnings from Madness, Rack and Honey by Mary Ruefle, then set the text using web coding so that the reader will not only be able to read the story but also experience my interpretation of the story.

This website directs the viewers to a maze, in which different choices lead to different routes, delivering a nonlinear reading experience. As the author mentioned, the poems have no end, so this maze will continue forever.
